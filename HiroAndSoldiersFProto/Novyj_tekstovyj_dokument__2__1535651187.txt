В некой игре-стратегии есть солдаты и герои. 

У солдат есть метод «идти за героем», который в качестве аргумента принимает объект типа «герой». 
У героев есть метод «увеличения собственного уровня».

В основной ветке программы создается по одному герою для каждой команды. 

В цикле генерируются солдаты. Их количество и принадлежность к команде определяется случайно. 

У героев есть метод расчет индекса силы который считается по формуле: 
индекс_силы = количество_солдат*уровень_героя

У героев есть метод «нападение» который принимает объект другого героя. 
Герой у которого больше индекс силы – побеждает, ему повышается уровень на одну единицу.
